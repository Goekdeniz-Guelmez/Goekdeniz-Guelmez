## Projects

![actively maintained](https://img.shields.io/badge/●_actively_maintained-2ea043?style=flat) ![maintained](https://img.shields.io/badge/●_maintained-d29922?style=flat) ![low_activity](https://img.shields.io/badge/●_low_activity-6e7681?style=flat)

**[mlx-lm-lora](https://github.com/Goekdeniz-Guelmez/mlx-lm-lora)** ![actively maintained](https://img.shields.io/badge/●-2ea043?style=flat) — LoRA / QLoRA / full fine-tuning on Apple Silicon. 12+ training methods, DPO / GRPO / ORPO / PPO, Muon optimizer, WandB. The go-to fine-tuning toolkit for M-series.

**[mlx-vlm](https://github.com/Blaizzy/mlx-vlm)** *(main trainer maintainer)* ![actively maintained](https://img.shields.io/badge/●-2ea043?style=flat) — Vision-language model training on MLX. Fully rewrote the training backend, added ORPO support.

**[MLX-Embeddings-LoRA](https://github.com/Goekdeniz-Guelmez/mlx-embeddings-lora)** ![actively maintained](https://img.shields.io/badge/●-2ea043?style=flat) — Fine-tune embedding models for retrieval and semantic tasks on Apple Silicon.

**[MLX-Benchmark](https://github.com/Goekdeniz-Guelmez/MLX-Benchmark)** ![actively maintained](https://img.shields.io/badge/●-2ea043?style=flat) — First CLI benchmark measuring LLM understanding of the MLX ecosystem and its APIs.

**[Moshi-FineTune-MLX](https://github.com/Goekdeniz-Guelmez/moshi-finetune-mlx)** ![actively maintained](https://img.shields.io/badge/●-2ea043?style=flat) — LoRA and full fine-tuning for Moshi speech-to-speech models on Apple Silicon.

**[Local NotebookLM](https://github.com/Goekdeniz-Guelmez/Local-NotebookLM)** ![actively maintained](https://img.shields.io/badge/●-2ea043?style=flat) — Fully local, PDF-grounded audio generation (up to 6 speakers). No API keys, no cloud. Companion [native app](https://github.com/Goekdeniz-Guelmez/Local-Notebook-LM-App) included.

**[MLX-LM-LENS](https://github.com/Goekdeniz-Guelmez/mlx-lm-lens)** ![maintained](https://img.shields.io/badge/●-d29922?style=flat) — Interpretability and abliteration tooling for MLX language models.

**[MLX-KAN](https://github.com/Goekdeniz-Guelmez/mlx-kan)** ![low_activity](https://img.shields.io/badge/●-6e7681?style=flat) — Kolmogorov-Arnold Networks, natively in MLX.

**[Gabliteration](https://github.com/Goekdeniz-Guelmez/gabliteration)** ![low_activity](https://img.shields.io/badge/●-6e7681?style=flat) — Companion repo to [arXiv:2412.06527](https://arxiv.org/abs/2412.06527). Remove refusal directions from any HF Transformers model.
